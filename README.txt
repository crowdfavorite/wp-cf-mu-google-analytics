# CF MU Google Analytics Plugin
Makes it possible (and easy) to add multiple Google Analytics Tracking Codes to your blog.  Also, it makes possible the ability to have one site-wide tracker, as well as a blog-specific tracker.

## Basic Usage
- Enable Plugin
- Visit the <a href="options-general.php?page=cf-mu-google-analytics.php">Plugin's Settings Page</a>
- Add the Blog's tracking code
- Click 'Add another tracking ID for this blog' if you have another tracking code for your blog.
- Site administrators will also see another input for adding of the site-wide tracking code.
